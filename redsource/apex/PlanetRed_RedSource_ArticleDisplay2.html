Steps


Handling SDG Queue


Step Name Description
1 Select your role (CPD Admin)
2 Please insert your Windows Login ID it’s a must
3 Once you are logged in, you will see 3 tabs / sections –
1. Technical Investigation Queue
2. TT work-list
3. Monitor
4 Technical Investigation Queue –
This queue will let you know how many cases needs to be actioned and TT needs to be raised via ITSM
5



TT work-list –
This section will help you to search the content such as below –
1. Case Status – Open/Technical investigation/Trouble Ticket / Closed
2. Case ID
3. MSISDN
4. TT ID
6 Monitor –
This section allows you to search data issue wise for date range
7 How do you action cases and create TT
 Open the case from the Technical Investigation Queue in SDG Queue Management
8 Click on Open to view the information about the case
9 Once the case is opened, You will have to insert the information from SDG Tool into ITSM Tool and then capture the TT ID and insert it into SDG Tool

Do not forget to capture the correct Call-back Date into the SDG Tool

Once you have verified the information which is inserted into both the tools please click on TT Raised Tab in the SDG Tool
